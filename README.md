A secure, command-line bidding application built as part of the 100 Days of Code: The Complete Python Pro Bootcamp. This project demonstrates the power of Python Dictionaries and the concept of "nesting" data.
🚀 Features
Secret Bidding: Multiple users can enter bids privately without others seeing the input.
Winner Logic: The program automatically calculates the highest bidder and announces the winner.
Dynamic Input: Handles any number of participants using a while loop.
🧠 What I Learned Today
Dictionaries: Using key-value pairs to link names (keys) to bid amounts (values).
Looping Through Dictionaries: Iterating through stored data to compare integers and find a maximum value.
Logic Flow: Controlling program execution with boolean flags (continue_bidding).
Git/GitHub Workflow: Successfully implemented version control, handled environment configuration (Git engine), and managed selective file staging
